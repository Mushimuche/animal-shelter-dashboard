# Left Behind — Long Beach Animal Shelter Dashboard

## Setup

```bash
pip install -r requirements.txt
```

## Run

Place `animal-shelter-intakes-and-outcomes.csv` in the **same folder** as `app.py`, then:

```bash
shiny run app.py
```

Open your browser at http://127.0.0.1:8000


## Filters (apply to all charts)
- **Animal Type** — All / CAT / DOG / BIRD / etc.
- **Intake Year** — All / 2017–2026
- **Intake Type** — All / STRAY / OWNER SURRENDER / etc.
